Digital Paper Evaluation

Exam papers or evaluated marks can be missed accidentally.
To solve this, we automated the process of evaluation of exam papers and storing of marks using
an online application.
This stores softcopy of exam papers and also allotted marks in the database which can be accessed
anytime and anywhere.
It has separate access according to the role of user in the college/Institute.
Evaluation of papers by evaluators from various places can be done online,
which saves time, distance problems and evaluator availability, travel alliances, etc.
